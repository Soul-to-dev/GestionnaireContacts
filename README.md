# Gestionnaire de Contacts - Java

## Description
Programme Java qui permet de gérer une liste de contacts.
On peut ajouter des contacts et rechercher un contact par son nom.

## Fonctionnalités
- Ajouter un contact (nom + téléphone)
- Rechercher un contact par son nom
- Afficher le numéro de téléphone d'un contact trouvé

## Structure du projet
- `Contact.java` : représente un contact avec un nom et un téléphone
- `ContactsManager.java` : gère la liste des contacts (ajout et recherche)
- `Main.java` : point d'entrée du programme, teste les fonctionnalités

## Comment exécuter
1. Cloner le projet : `git clone https://github.com/Soul-to-dev/GestionnaireContacts.git`
2. Ouvrir dans IntelliJ IDEA
3. Lancer `Main.java`
